This folder includes all STL files that are ready to be printed. Most are exported y-axis up, but reorient them as you see fit.
Recommended print settings:
Material: PETG is okay, ABS/ASA highly recommended
Perimeters/walls: 4
Infill: 100%
Top/bottom layers: 3
Print speed: no greater than 200mm/s

Process for strengthening using epoxy/super glue coming soon.
